My journey so far

From equations to production systems.

I started with pure maths & physics at Faculté des Sciences et Techniques in Fès because I loved solving problems that look impossible at first sight.

That led to a scholarship at UPF, where I dove head-first into computer engineering and realized I liked designing systems as much as solving equations.

Today I'm a 4th-year IATIC student at UVSQ / Université Paris-Saclay, living between France and Morocco, and building software that has to work in the real world.

I enjoy taking vague requirements and messy constraints and turning them into clean architectures, secure APIs and dashboards people actually like using.

My current obsessions:

  • Backend architectures that are still readable 6 months later
  • Real-time computer vision pipelines for stadium surveillance
  • Event-driven systems with proper error handling (no more silent failures)

Network projects · Telecom

Huawei Tracker

Internal project tracking system for telecom network rollouts for the IAM NPM team at Huawei. Handles hundreds of projects with different teams and access levels.

  • Spring Boot · Angular · MongoDB · JWT · Docker
  • Role-based access control & multi-level permissions
  • Progress tracking, automated reporting & dashboards

The interesting part: designing flexible data models and access rules that match how people actually work, not just the ideal spec.

Computer vision · Security

Stadium Violence Detection System

Real-time CCTV analysis that spots violent behaviours in football stadiums and triggers alerts before things escalate.

  • Python · YOLOv8 · LSTM · RabbitMQ · Docker
  • Angular dashboard & Spring Boot backend
  • Designed for low latency across multiple camera feeds

Lesson learned: production ML is 20% training models and 80% handling edge cases, latency and weird data.

Reporting · Data modeling

Dynamic Reporting Platform

A flexible reporting system where admins define report templates and sheets, and users generate complex reports without schema migrations.

  • Spring Boot · MongoDB
  • Embedded templates & dynamic sheet structures
  • Designed around how real project teams report

Built to explore how far we can push flexibility while keeping queries fast and code maintainable.
